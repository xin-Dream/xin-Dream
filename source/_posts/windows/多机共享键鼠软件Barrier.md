当操作多台设备时，可以无缝切换键盘鼠标，同时可以复制内容

[windows下载地址](https://github.com/debauchee/barrier/releases)


Linux安装
```bash
sudo apt-get install barrier
```

安装结束后，选择一台设备作为服务端，其他作为客户端即可

![1](../../images/多机共享键鼠软件Barrier/1.png)

在设置服务端按钮下可以添加屏幕布局
